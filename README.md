## ğŸ¨ Customization

### Adding Your Profile Picture

1. Add your profile picture as `profile.jpg` in the root directory
2. Recommended size: 500x500px or larger (square ratio)
3. If no image is provided, a gradient placeholder with initials will be shown

### Updating Publication Links

Edit `index.html` and replace `#` in publication links with actual URLs:

```html
<a href="YOUR_PDF_URL" class="pub-link"><i class="fas fa-file-pdf"></i> PDF</a>
<a href="YOUR_GITHUB_URL" class="pub-link"><i class="fab fa-github"></i> Code</a>
```

### Adding Your CV

1. Place your CV PDF as `CV_wooseong.pdf` in the root directory
2. Or update the link in `index.html`:
   ```html
   <a href="YOUR_CV_FILENAME.pdf" target="_blank" title="CV">
   ```

### Changing Colors

Edit CSS variables in `styles.css`:

```css
:root {
    --accent-primary: #2563eb;  /* Main accent color */
    --accent-secondary: #3b82f6;
    /* ... */
}
```
